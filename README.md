# Git/GitHubを学ぼう！！

## Gitのコマンド

リポジトリの初期化

```
git init
```

リポジトリの状態を確認する方法

```
git status
```

gitにファイルを追加 (ステージング)

```
git add ファイル名
```

ステージングされた内容のcommit

```
git commit -m "commitメッセージ"
```

ファイルの差分の確認

```
git diff
```

バージョン履歴の確認

```
git log
```

commitで行った変更を確認 (git log でコミットIDが確認できる)

```
git show コミットID
```

branchの確認

```
git branch
```

branchの作成と移動
オプション '-b' をつけると新規で作成される
オプションがなければ，そのbranchへと移動する

```
git branch -b branch名
```
